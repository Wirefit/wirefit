# wirefit

**Language-agnostic contract checking.**

This npm package is a **placeholder** reserving the `wirefit` name. The tool currently
ships as a Go binary and GitHub Action. A packaged npm distribution (and the TypeScript
extractor) is planned.

- Project & docs: https://github.com/wirefit/wirefit
- License: Apache-2.0

Install today (Go):

```sh
go install github.com/wirefit/wirefit/cmd/wirefit@latest
```
